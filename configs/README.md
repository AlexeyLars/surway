# Configuration Files

Эта папка содержит конфигурационные файлы для различных компонентов приложения.

## Файлы конфигурации

### redis.conf
Конфигурация Redis для продакшен среды с настройками персистентности, управления памятью, производительности и безопасности.

### frontend.conf
Конфигурация фронтенда приложения, включая:
- API endpoint (протокол, хост, порт, версия)
- Frontend URL (протокол, хост, порт)

**Использование:**
1. Скопируйте значения из `frontend.conf` в файл `.env.local` в папке `frontend/`
2. Используйте формат: `NEXT_PUBLIC_API_PROTOCOL=http` и т.д.
3. Для примера см. `frontend/env.example`
